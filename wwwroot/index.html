Velkommen forsvarets web API

Endepunkter

Soldier

  • GET - få tak i alle Soldater: /Soldier
  • POST - lagre soldat: /Soldier
  • PUT - Endre soldat: /Soldier
  • POST - lagre bilde: /ImageUpload/PostImage
  • DELETE - slette soldat: /Soldier/{id}
  • GET - får tak i soldater fra avdeling: /Soldier/GetDepartment/{department}

Fartøy

  • GET - få tak i alle fartøy: /Vehicle
  • POST - lagre fartøy: /Vehicle
  • DELETE - slette fartøy: /Vehicle/{id}
  • GET - får tak i fartøy fra kategori: /Soldier/GetCategory/{Category}

Weapon

  • GET - få tak i alle Våpen: /Weapon
  • POST - lagre Våpen: /Weapon
  • DELETE - slette fartøy: /Weapon/{id}
  • GET - får tak i våpen på Type: /Weapon/GetType/{typey}
